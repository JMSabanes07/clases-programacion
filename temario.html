Temario de Programación con JavaScript

1. Introducción a la Programación y JavaScript

¿Qué es la programación?

La programación es el proceso de diseñar y construir un conjunto de instrucciones que una computadora puede interpretar para realizar tareas específicas. Es la base de todas las aplicaciones y sistemas que utilizamos a diario.

¿Cómo funcionan los lenguajes de programación?

Un lenguaje de programación sirve como puente entre los humanos y las máquinas, permitiéndonos escribir instrucciones que las computadoras pueden entender. Se clasifican en:

  • Lenguajes de bajo nivel: Más cerca del lenguaje de máquina, como Assembly. Son rápidos pero difíciles de escribir y leer.
  • Lenguajes de alto nivel: Más intuitivos y fáciles de usar, como JavaScript y Python, ideales para desarrollar aplicaciones complejas con menos esfuerzo.

Compiladores vs. Intérpretes

Los programas escritos en lenguajes de programación necesitan ser convertidos a lenguaje máquina. Esto se logra a través de:

  • Compiladores: Traducen todo el código a lenguaje máquina antes de ejecutarlo, generando un archivo ejecutable.
  • Intérpretes: Traducen y ejecutan el código línea por línea, útil para pruebas rápidas y lenguajes dinámicos como JavaScript.

Instalación de Node.js y configuración del entorno

Node.js es un entorno que permite ejecutar JavaScript fuera del navegador. Para instalarlo: 


# Descargar e instalar Node.js desde https://nodejs.org
node -v  # Ver versión de Node.js
npm -v   # Ver versión de npm

2. Variables

¿Qué son las variables?

Las variables son contenedores para almacenar datos en memoria, que pueden cambiar durante la ejecución de un programa. En JavaScript, se pueden declarar usando var, let, o const.

Declaración de variables

La declaración de variables puede hacerse de las siguientes maneras:


// Usando var (antiguo)
var nombre = "Juan";

// Usando let (recomendado para variables que cambian)
let edad = 25;

// Usando const (para valores constantes)
const PI = 3.1416;

Tipos de datos

JavaScript soporta diferentes tipos de datos, como:

  • Números: let numero = 42;
  • Cadenas: let texto = "Hola";
  • Booleanos: let esVerdadero = true;
  • Objetos: let persona = { nombre: "Ana", edad: 30 };
  • Arreglos: let colores = ["rojo", "verde", "azul"];

3. Estructuras de Control

Condicionales

Los condicionales permiten tomar decisiones en el programa según ciertas condiciones. Por ejemplo: 


let edad = 18;
if (edad >= 18) {
  console.log("Eres mayor de edad");
} else {
  console.log("Eres menor de edad");
}

Ciclos

Los ciclos se utilizan para ejecutar un bloque de código varias veces. Existen varios tipos, como for, while y do-while: 


for (let i = 0; i < 5; i++) {
  console.log(`Número: ${i}`);
}

Uso de break y continue

Dentro de los ciclos, break detiene la ejecución del ciclo, mientras que continue salta a la siguiente iteración. 


for (let i = 0; i < 10; i++) {
  if (i === 5) break;
  console.log(i);
}

Ejercicios:

  • Crea un ciclo que imprima los números del 1 al 10 en la consola.
  • Modifica el ciclo anterior para que salte el número 5 usando continue.

4. Funciones

Declaración y ejecución de funciones

Las funciones son bloques de código reutilizables que realizan una tarea específica. Aquí un ejemplo básico: 


function saludar() {
  console.log("Hola, mundo!");
}
saludar();

Funciones anónimas

Una función anónima es aquella que no tiene nombre y se asigna a una variable para ser reutilizada: 


const multiplicar = function (a, b) {
  return a * b;
};
console.log(multiplicar(4, 6));

Funciones flecha

Las funciones flecha son una forma más compacta de escribir funciones en JavaScript:


const dividir = (a, b) => a / b;
console.log(dividir(10, 2));

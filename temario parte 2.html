5. Objetos y Clases

Objetos en JavaScript

Un objeto es una colección de propiedades y métodos. Las propiedades son valores asociados a un objeto, mientras que los métodos son funciones que se pueden ejecutar dentro del objeto.


const persona = {
  nombre: "Juan",
  edad: 30,
  saludar: function() {
    console.log(`Hola, mi nombre es ${this.nombre}`);
  }
};
persona.saludar();

Clases en JavaScript

Las clases son plantillas para crear objetos con propiedades y métodos definidos.


class Animal {
  constructor(nombre, tipo) {
    this.nombre = nombre;
    this.tipo = tipo;
  }

  describir() {
    return `${this.nombre} es un ${this.tipo}`;
  }
}
const gato = new Animal("Michi", "gato");
console.log(gato.describir());

Ejercicios:

  • Crea un objeto que represente un coche con propiedades como marca, modelo y un método para arrancar.
  • Escribe una clase para representar un rectángulo con métodos para calcular el área y el perímetro.
  • Crea una instancia de la clase que definiste y muestra sus propiedades y métodos en la consola.

6. Promesas y Async/Await

Introducción a Promesas

Las promesas son objetos que representan la eventual finalización (o falla) de una operación asíncrona.


const promesa = new Promise((resolve, reject) => {
  const exito = true;

  if (exito) {
    resolve("Operación exitosa");
  } else {
    reject("Error en la operación");
  }
});

promesa.then((mensaje) => console.log(mensaje)).catch((error) => console.error(error));

Async/Await

Async/Await es una forma más clara y concisa de trabajar con operaciones asíncronas, basada en promesas.


async function obtenerDatos() {
  try {
    const respuesta = await fetch("https://api.example.com/datos");
    const datos = await respuesta.json();
    console.log(datos);
  } catch (error) {
    console.error("Error al obtener datos", error);
  }
}
obtenerDatos();

Ejercicios:

  • Escribe una promesa que simule una operación de carga de datos con un setTimeout.
  • Implementa una función asíncrona que haga una llamada ficticia a una API usando fetch.
  • Crea un ejemplo donde combines Promesas y Async/Await para manejar múltiples operaciones asíncronas.

7. Manipulación del DOM

Seleccionar Elementos

Podemos seleccionar elementos del DOM usando métodos como querySelector y getElementById.


const encabezado = document.querySelector("h1");
console.log(encabezado.textContent);

Modificar Contenido

Es posible cambiar el contenido y los estilos de los elementos.


const parrafo = document.querySelector("p");
parrafo.textContent = "Este es un texto modificado.";
parrafo.style.color = "blue";

Eventos

Podemos agregar interactividad con eventos como click, mouseover, etc.


const boton = document.querySelector("button");

boton.addEventListener("click", () => {
  alert("¡Botón clickeado!");
});

Ejercicios:

  • Crea un botón que cambie el color de fondo del documento al hacer clic.
  • Escribe un programa que permita añadir elementos a una lista desde un campo de texto.
  • Implementa un contador que se incremente al hacer clic en un botón.
